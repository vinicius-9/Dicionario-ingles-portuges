## ⚠️ Windows bloqueou o aplicativo?

### Opção 1
1. Clique em **Mais informações**
2. Clique em **Executar assim mesmo**

### Opção 2
1. Clique com o botão direito no arquivo `.exe`
2. Abra **Propriedades**
3. Marque **Desbloquear**
4. Clique em **Aplicar** e depois em **OK**

## ✨ Funcionalidades

- Cadastro de palavras com significado e exemplo
- Busca rápida por palavra ou significado
- 60 palavras pré-cadastradas para já começar estudando
- Dados salvos localmente — sem internet, sem cadastro

## 🛠️ Como compilar

1. Baixe ou clone este repositório
2. No terminal, dentro da pasta, rode:  dotnet run


**Requisitos para compilar:**
- Windows 10/11
- .NET 8 SDK *(o build.bat instala automaticamente se não tiver)*
